Smartly switches between VK and YouTube videos based on user geolocation. Adds a button for the Classic Editor and a block for Gutenberg.

== Description ==
Автоматически подставляет видео с YouTube или VK в зависимости от геолокации пользователя. Удобная кнопка для классического редактора и блок для Gutenberg.

== Installation ==
1. Загрузите архив с плагином через меню "Плагины > Добавить новый > Загрузить плагин".
2. Активируйте плагин через меню "Плагины" в WordPress.
